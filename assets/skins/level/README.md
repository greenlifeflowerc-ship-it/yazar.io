# Level skins

Drop `.png`, `.jpg`, `.jpeg`, or `.webp` files into this folder. They will
appear automatically in the **Level** tab of the in-game skin chooser the
next time the app runs.

Tips:
- Square images work best — they are clipped to a circle when shown on a cell.
- Transparent backgrounds (PNG with alpha) compose nicely over the cell colour.
- File names are not shown anywhere, so anything readable is fine.
